Formatos para crear Github pages

  1. Por medio del archivo Readme.md
  2. Por medio de un archivo HTML personalizado llamado index.html

Para que la página sea por medio de Readme.md (texto markdown), lo único que se requiere es un archivo con el nombre Readme.md con el contenido de la página.
Si se opta por usar un Readme.md, es posible seleccionar Themes de Github.

Para que la página sea por medio de HTML personalizado, se requiere la existencia de un archivo index.html, y NO DEBE DE EXISTIR ningún archivo con el nombre "Readme.md". Caso que exista, Github reconocerá y utilizará al archivo "Readme.md". Si se opta por usar un archivo HTML personalizado index.html, los Themes de Github no serán aplicados.
